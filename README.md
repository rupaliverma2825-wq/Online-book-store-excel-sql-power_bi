# Online-book-store-excel-sql-power_bi


1) Business Intelligence & Analytics for Online Bookstore-

Online Book Store Data Analytics is a data-driven project aimed at analyzing customer purchases, book inventory, and sales trends for an online bookstore. With the increasing growth of e-commerce in the publishing industry, understanding customer preferences, high-selling book categories, and revenue patterns is essential for improving business performance and customer satisfaction.
This project uses a dataset that includes customer details, book information, and order transactions to extract meaningful insights for strategic decision-making.
